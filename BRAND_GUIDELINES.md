## Brand Overview

### Mission Statement
**BideshGomon** empowers Bangladeshi citizens to explore global opportunities through transparent visa application services, comprehensive migration support, and professional career guidance.

### Brand Positioning
- **Category:** Digital Migration & Visa Services Platform
- **Target Audience:** Bangladeshi professionals, students, and families seeking international opportunities
- **Geographic Focus:** Bangladesh (Primary), South Asia (Secondary), Global (Tertiary)
- **Brand Personality:** Trustworthy, Professional, Accessible, Empowering

### Brand Promise
*"Your trusted partner in making international dreams a reality - transparent, efficient, and always by your side."*

### Core Values
1. **Transparency** - Clear processes, honest pricing, no hidden fees
2. **Reliability** - Consistent service delivery, dependable support
3. **Accessibility** - User-friendly platform, multilingual support (Bengali/English)
4. **Empowerment** - Education-focused, skill development, career growth
5. **Integrity** - Ethical practices, regulatory compliance, data security

```

- Change logo colors outside approved palette
- Add effects (shadows, gradients, outlines)

- Place on busy backgrounds without container
- Use outdated logo versions
- Recreate or modify logo elements

#### Hierarchy

#### Combinations

**✅ APPROVED:**
- White background + Red primary + Gray text + Green accents
- Gray background + White cards + Red CTAs + Green success

**❌ NOT APPROVED:**
- Red + Green together (avoid color blindness issues)
- Blue primary buttons (only accent/icons allowed)
- Multiple bright colors in same section
- Colored text on colored backgrounds

- **Normal Text:** Minimum 4.5:1 contrast ratio
- **Large Text (18pt+):** Minimum 3:1 contrast ratio
- **Interactive Elements:** Minimum 3:1 against backgrounds

```
Headings:    1.2 (120%) - Tight for impact
Body:        1.5 (150%) - Comfortable reading

```

### Letter Spacing (Tracking)
```
Headlines:   -0.02em - Tight for visual impact
Headings:    -0.01em - Slightly tight
Body:        0em     - Default spacing
All Caps:    +0.05em - Increased for readability
```

#### Bengali Weights

```

#### Bengali Type Scale
```

### Type Pairing Rules

**✅ DO:**
- Pair Inter (Latin) with Noto Sans Bengali
- Use consistent weights across languages
- Maintain same size for equivalent hierarchy
- Keep line heights generous for Bengali (1.6x)

**❌ DON'T:**
- Mix more than 2 font families
- Use decorative fonts for body text
- Set Bengali text below 14px
- Use compressed/condensed fonts

## Visual Identity

### Design Principles

#### 1. Simplicity
- Clean layouts with generous white space
- Maximum 3 colors per section
- Minimal decorative elements
- Clear visual hierarchy

#### 2. Consistency
- Reuse components across pages
- Maintain spacing system (8px grid)
- Standardize border-radius (8px standard, 4px small, 12px large)
- Uniform shadow system

#### 3. Accessibility
- WCAG 2.1 Level AA compliance minimum
- Color-blind friendly combinations
- Keyboard navigation support
- Screen reader optimization

#### 4. Responsiveness
- Mobile-first design approach
- Breakpoints: 375px, 768px, 1024px, 1280px, 1536px
- Touch-friendly targets (minimum 44px × 44px)
- Fluid typography and spacing

### Spacing System

**8px Base Grid:**
```
xs:  4px   (0.25rem) - Tight elements
sm:  8px   (0.5rem)  - Compact spacing
md:  16px  (1rem)    - Standard spacing
lg:  24px  (1.5rem)  - Section spacing
xl:  32px  (2rem)    - Large sections
2xl: 48px  (3rem)    - Page sections
3xl: 64px  (4rem)    - Hero sections
```

**Application:**
- Component padding: 16px (md)
- Card spacing: 24px (lg)
- Section spacing: 48px (2xl)
- Hero spacing: 64px (3xl)

### Border Radius

```

## Brand Voice & Messaging

### Brand Voice Attributes

#### Professional Yet Approachable
- **Tone:** Confident but not arrogant
- **Language:** Clear, jargon-free, educational
- **Style:** Conversational yet expert
- **Example:** "We'll guide you through every step" vs. "We will facilitate your migration process"

#### Empowering & Supportive
- **Tone:** Encouraging, optimistic, motivating
- **Language:** Action-oriented, positive, inclusive
- **Style:** Partner mindset, not service provider
- **Example:** "Let's start your journey together" vs. "Submit your application here"

#### Transparent & Trustworthy
- **Tone:** Honest, straightforward, reliable
- **Language:** Clear expectations, no hidden info
- **Style:** Upfront about processes, timelines, costs
- **Example:** "Average processing time: 4-6 weeks" vs. "Fast processing available"

#### Culturally Sensitive
- **Tone:** Respectful, inclusive, aware
- **Language:** Bengali + English, local context
- **Style:** Understand family dynamics, financial situations
- **Example:** "We understand family is priority" vs. Generic corporate speak

### Writing Style Guide

#### Headlines
```
✓ Clear, benefit-focused: "Get Your Student Visa in 60 Days"
✓ Action-oriented: "Start Your Canadian Journey Today"
✓ Specific numbers: "Join 10,000+ Successful Applicants"
✗ Vague: "Best Visa Service"
✗ Jargon: "Leveraging Our Migration Ecosystem"
```

#### Body Copy
```
✓ Short sentences (15-20 words max)
✓ Active voice: "We process applications quickly"
✓ You-focused: "Your documents are secure with us"
✗ Passive voice: "Applications are processed by our team"
✗ Long paragraphs (keep to 3-4 lines)
```

#### Calls-to-Action (CTAs)
```
✓ Action verbs: "Start Application", "Book Consultation"
✓ Value-driven: "Get Free Assessment"
✓ Urgency (when appropriate): "Apply Before Deadline"
✗ Generic: "Click Here", "Learn More", "Submit"
```

#### Error Messages
```
✓ Helpful: "Email format incorrect. Try: name@example.com"
✓ Polite: "Oops! Please check your password (min 8 characters)"
✗ Blame: "You entered invalid data"
✗ Technical: "Error 422: Validation failed"
```

## Photography & Imagery

### Photography Style

#### Subject Matter
**✓ Preferred:**
- Authentic Bangladeshi people (diverse ages, backgrounds)
- Real office environments (clean, professional)
- Genuine success stories (with permission)
- Cultural landmarks (Bangladesh and destinations)
- Documentation process (respectful, not intrusive)

**✗ Avoid:**
- Generic stock photos (Western-centric)
- Staged, overly-posed scenarios
- Controversial political imagery
- Distressed or negative situations
- Outdated technology/fashion

#### Composition
- **Rule of Thirds:** Primary subjects off-center
- **Eye Level:** Natural perspective, engaging
- **Depth:** Shallow depth of field for portraits
- **Lighting:** Natural light preferred, warm tones
- **Colors:** Match brand palette in post-processing

#### Technical Specs
```
Resolution:  300 DPI (print), 72 DPI (web)
Format:      JPEG (web), TIFF (print)
Color:       sRGB (web), CMYK (print)
Size:        2400px wide minimum (web hero images)
```

### Image Treatment

#### Color Grading
```css
/* Subtle warm tone */
Temperature:  +5 (slightly warm)
Tint:         0 (neutral)
Saturation:   +10 (enhanced, not oversaturated)
Contrast:     +15 (crisp but natural)
```

#### Filters
**Approved Presets:**
1. **Warm Professional** - Slight warmth, increased clarity
2. **Natural Bright** - Lifted shadows, white balance
3. **Cultural Vibrant** - Enhanced colors for cultural events

**❌ Avoid:**
- High-contrast black & white
- Heavy vignetting
- Overly saturated colors
- Instagram-style heavy filters

### Illustration Style

#### When to Use
- Explaining complex processes
- Data visualization
- Icon systems
- Empty states, error pages
- Loading animations

#### Style Guidelines
```
Line Weight:    2-3px
Colors:         Brand palette only
Style:          Flat design, minimal gradients
Corners:        Rounded (8px radius)
Perspective:    Isometric or flat frontal
Detail:         Simple, not overly detailed
```

#### Illustration Library

### Icon Usage Rules

**✓ DO:**
- Use consistent stroke width
- Maintain optical balance
- Align icons to pixel grid
- Export in multiple sizes (16px, 24px, 32px)
- Provide alt text for accessibility

**✗ DON'T:**
- Mix outline and filled icons
- Use decorative icons without purpose
- Resize icons non-proportionally
- Use icons below 16px size
- Forget accessible labels
